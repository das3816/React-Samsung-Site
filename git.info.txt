ls - показывает название файлов при переходе cd
git clone <ссылка на репозиторий> - клонирует репозиторий
git status - показывает статус файлов(сохранены они или нет)
git add <название файла> - следит за изменениями файла
git commit -m "first commit" - сохраняет изменения в файле
git push <ссылка на репозиторий> - посылает данные на сервер
